# Furnifuture-Furniture Website
Description:
Furnifuture is a simple and modern furniture showcase website built using HTML and CSS.
The website presents different furniture items in a clean and visually appealing layout, allowing users to explore furniture products easily.

Features:
- Modern homepage design
- Navigation bar with Home, About us and Contact sectons
- Furniture product showcase
- Provided ratings and reviews

Technologies Used:
- HTML
- CSS
